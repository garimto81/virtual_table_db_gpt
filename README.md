## 🛠️ 핵심 기능 워크플로우

### Hand 시트와 Virtual 시트 관계
```mermaid
graph LR
    A[현장 플레이] --> B[Hand 시트 기록]
    B --> C{중요한 핸드?}
    C -->|Yes| D[Virtual 시트 수동 등록]
    C -->|No| E[등록 안함]
    D --> F[CSV 게시]
    F --> G[웹 앱 분석]
```

### ⚠️ 중요 원칙
- **Hand 시트**: 모든 핸드 자동 기록 (전체 데이터)
- **Virtual 시트**: 선별된 핸드만 수동 등록 (분석 대상)
- **자동 동기화 금지**: Hand → Virtual 자동 복사 ❌
